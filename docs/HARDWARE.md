# 硬件接线说明

## 组件清单

| 组件         | 型号               | 数量 | 备注               |
| ------------ | ------------------ | ---- | ------------------ |
| 主控板       | ESP32-S3-DevKitC-1 | 1    | 3.3V IO            |
| CO2传感器    | ZG09SR             | 1    | NDIR红外，UART接口 |
| 气泵         | 5V微型气泵         | 1    | 用于气路采样       |
| 过滤瓶       | -                  | 1    | 干燥管/除湿        |
| 密封盒       | 气体超密封盒       | 1    | 放置被测样品       |
| 微型气室     | 铝箔材质           | 1    | 包裹传感器         |
| 反倒吸装置   | -                  | 1    | 防水倒吸（藻类环境有水） |

## ZG09SR 传感器

ZG09SR 是深圳盛世物联生产的 NDIR (非色散红外) CO2 传感器：

- 测量范围: 0-5000ppm (可选更高量程)
- 精度: ±(50ppm + 5%读数)
- 接口: UART (9600bps, 8N1)
- 工作电压: 4.5-5.5V
- UART 信号兼容 3.3V

## 电源说明

**方案一：单电源 (USB 5V)**

```
USB 5V ──┬── ESP32-S3 (内部3.3V稳压)
         │
         ├── ZG09SR (5V)
         │
         └── 气泵 (5V，经 MOS 控制)
```

**方案二：双电源 (3V + 5V)** ※ 本项目实际采用

```
3V 电源 ── ESP32-S3
5V 电源 ──┬── ZG09SR
          └── 气泵 (经 MOS 芯片控制)
```

- ESP32-S3：3.3V IO，可用 3V 直接供电或 USB 5V 内部稳压
- ZG09SR：需 5V 供电，UART 信号 3.3V 兼容，可直连 ESP32
- 气泵：5V 供电，通过 MOS 芯片由 GPIO 控制开关

## 引脚连接

### ZG09SR CO2 传感器 (UART)

| ESP32-S3 | ZG09SR | 说明       |
| -------- | ------ | ---------- |
| GPIO 16  | TXD    | 传感器发送 |
| GPIO 17  | RXD    | 传感器接收 |
| 5V       | VCC    | 电源       |
| GND      | GND    | 地         |

### 气泵 (经 MOS 芯片控制)

| ESP32-S3 | MOS 芯片 | 气泵 | 说明           |
| -------- | -------- | ---- | -------------- |
| GPIO 18  | 栅极 G   | -    | PWM 控制信号   |
| -        | 漏极 D   | 气泵 | 5V 经 MOS 到气泵 |
| GND      | 源极 S   | GND  | 共地           |

> ESP32 的 3.3V GPIO 不能直接驱动 5V 气泵，需通过 MOS 芯片做开关控制。

## 接线示意

```
                    ESP32-S3-DevKitC-1
                   ┌─────────────────┐
                   │  3V 或 USB 供电  │
                   │                 │
   ZG09SR TXD ───► │ GPIO16 (RX)     │
   ZG09SR RXD ◄─── │ GPIO17 (TX)     │
   MOS 栅极   ◄─── │ GPIO18 (PWM)    │  气泵控制
                   │                 │
                   └─────────────────┘
   (ZG09SR、气泵 由 5V 电源供电)
```

## 气路设计

本项目采用的气路布置（气体密封盒内含气泵与传感器）：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  进气管 → 过滤装置 → 气体密封盒 ─────────────────────────────────────────────│
│           (干燥除湿)    │                                                    │
│                        │ 内含：气泵、检测气室(传感器密封在内)                 │
│                        │                                                    │
│                        ▼                                                    │
│  气体流向： 过滤后进气 → 气泵抽气 → 检测气室(传感器读数) → 反倒吸 → 出密封盒  │
│                        (负压)      (铝箔微型气室)       (防水倒吸)            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 各部件功能

| 部件             | 功能                                           |
| ---------------- | ---------------------------------------------- |
| 过滤装置（干燥管） | 除湿，防止水汽影响传感器                       |
| 气体超密封盒     | 放置被测样品（如藻类培养物），内含气泵、检测气室 |
| 气泵             | 抽取密封盒内气体，送入检测气室                 |
| 检测气室（铝箔） | 包裹传感器，形成小体积检测腔，传感器读数处密封在内 |
| 反倒吸装置       | 防止关闭气泵时水/培养液倒吸进入气路（藻类环境有水） |

### 设计说明

- 气泵与传感器均在密封盒内，气泵抽气使气体流经检测气室
- 铝箔气室缩小检测体积，提高响应速度
- 反倒吸装置主要防止藻类培养液倒吸，兼防气体回流

### 关于漏气

即使使用"超密封盒"，长时间运行仍可能存在微小漏气，导致 CO2 数据持续下降。这正是本项目开发**漂移补偿算法**的原因——通过软件修正弥补硬件密封的不足。

## 自定义引脚

如需修改引脚分配，编辑 `firmware/src/main.cpp`（引脚在代码中定义）:

```cpp
// ZG09SR UART
#define RX_PIN  16   // 接传感器 TXD
#define TX_PIN  17   // 接传感器 RXD

// 气泵 PWM (经 MOS 芯片控制)
#define PUMP_PIN 18  // 接 MOS 栅极
```
